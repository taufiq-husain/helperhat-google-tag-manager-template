# HelperHat Live Chat Google Tag Manager Template

This repository contains the HelperHat Live Chat community template for Google Tag Manager.

The template loads the HelperHat install script on pages where the GTM tag fires:

```text
https://helperhat.com/install.js?businessId=YOUR_BUSINESS_ID
```

## Setup

1. Add the HelperHat Live Chat template to your Google Tag Manager web container.
2. Create a new tag from the HelperHat Live Chat template.
3. Paste your HelperHat business ID.
4. Set the trigger to All Pages or the pages where HelperHat should appear.
5. Preview the container and publish when the HelperHat widget appears correctly.

## Finding your business ID

Open HelperHat Settings and copy the value after `businessId=` in your install code.
